In questa pagina cercherò di raccontarti un pò più di me

Cosa so fare:

  1. Come UX/UI Designer
    • So fare analisi di mercato e della concorrenza
    • So intervistare gli utenti e definire le user personas: le loro caratteristiche, i pain points ed individuare le diverse customer journeys
    • So tradurre il tutto in flussi, diagrammi e sitemap per indivisuare le aree di miglioramento e le nuove opportunità
    • So creare wireframes e prototipi interattivi, da far testare agli utenti
    • So ripartire da capo
  2. Come Product Owner
    • So lavorare con metodologia agile e sovraintendere le cerimonie scrum
    • So definire requisiti di prodotto e user stories
    • So ascoltare ed adattarmi alle esigenze che cambiano
    • So fare la QA delle attività sviluppato per verificare la corrispondenza agli acceptance criteria
    • Sto imparando a dare le brutte notizie (work in progress...)
  3. Come Front-End Developer
    • Mi ricorderò come funzionano Html e Css e li conoscerò meglio
    • Scoprirò Javascript
    • Farò amicizia con React e l'AI
